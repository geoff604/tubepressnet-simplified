== Change Log file ==

Changes in 4.2.0

* Change to client-side auth to avoid storing secret on server
* Rewrote this Wordpress plugin so that it uses OAUTH2 request from the browser,
so no client secret will need to be stored on the web server.

Changes in 4.0 (Simplified fork)

* Make it work with Youtube Data API (v3)
* Remove unneeded functionality for now (now it will only work
  with importing video by id, other functionality could be added
  back later.
